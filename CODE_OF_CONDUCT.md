# Code of Conduct

Be respectful, direct, and constructive.

This project is intended to help people build safer MCP knowledgebase servers. Contributions should focus on technical merit, clear communication, and practical usefulness.

Unacceptable behavior includes harassment, discriminatory language, deliberate disruption, credential disclosure, or publishing private project content without consent.

Maintainers may remove comments, close issues, or block contributors who violate these expectations.
